About HTML

What does HTML stand for? HTML stands for Hyper Text Markup Language. HTML is the base language used for every single website online.

Who Invented it? Why? A man named Tim Berners-Lee invented HTML in 1993. Mr. Berners-Lee originally made it so that researchers could share their projects with other researchers and the rest of the world.

How HTML Websites are Published/Hosted Go to any writing tool like Google Docs, Word, or Notepad, type up the code, name it whatever you want, and at the end of the name type .html, and then either buy a domain or host your site for free on Github, Codepen, or any other hosting website, or dont host it at all and leave it as a file. You can't open an HTML file on a Chromebook, so if you make the file on a Chromebook you must either buy a domain, or host it free on the websites previously listed.
